# 支部会议记录整理

将用户提供的原始会议记录，按机电支部工作手册模板格式整理，聊天框直接输出。

## 输出结构

```
# 机电学院第X党支部 202X年X月 党组织生活记录
主题：XXX
信息表（时间/地点/参会人员/请假人员/列席人员/主持/记录）
会议主要议题（编号，只列实质性议题）
会议主要内容（按议题逐条展开 + 支部书记总结）
签字栏
```

## 核心规则

### 议题栏
- 只列**实质性议题**（学习、讨论、表决、工作部署等）
- **不写**"支部书记总结"
- 党费收缴列入议题即可

### 正文
| 内容类型 | 处理 |
|----------|------|
| 学习类议题 | 主持人引导语 + 选2-3位党员发言 + 末位总结句 |
| 党务工作类 | 概括说明，可分（一）（二）（三） |
| 党费收缴 | 议题列出，正文一句带过，不展开 |
| 主题党日部署 | 逐条列出必选项目内容 |
| 党员发言 | 选3-4位代表性发言，保留姓名+核心观点 |
| 支部书记总结 | 正文末尾保留，简要概括即可，**不一定需要提要求** |

### 格式
- 标题：方正小标宋，22号，居中
- 主题：楷体，15号，居中
- 信息表：仿宋，14号，表格式
- 小标题（一、二、）：楷体加粗，14号
- 正文：仿宋，14号，首行缩进
- 党员发言：仿宋，13号，左缩进，引用体
- 签字栏：右对齐

## 整理原则
- 保持原文关键信息，适当精简口语化表达
- 参会人员：原样列出
- 请假/列席：有则写，无则写"无"
- 主持：一般为支部书记
- 记录：实际记录人

## 示例

用户提供原始会议通知/记录文件后，直接输出整理后的完整记录文本。
