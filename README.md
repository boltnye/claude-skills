# Claude Code Skills

个人 Claude Code 技能集合，提升日常工作效率。

---

## 技能列表

### 1. party-meeting-record — 支部会议记录整理

将基层党组织会议原始记录（通知文件/逐字稿）整理为支部工作手册标准格式，直接输出到聊天框。

**适用场景：** 高校学生党支部月度党组织生活记录、支委会记录、民主评议记录等。

**输入：** 上级下发的会议通知 .docx、原始会议记录 .docx，或逐字稿文本。

**输出规则：**
- 标题 + 主题 + 信息表（时间/地点/参会人员/请假/列席/主持/记录）
- 会议议题（只列实质性议题）
- 会议主要内容（按议题逐条展开）
- 党员发言选取 3-4 位代表性发言
- 支部书记总结（简要概括）
- 签字栏

**触发词：** 会议记录、支部会议、整理记录、党组织生活记录

---

### 2. industry-handbook — 产业认知手册生成

按照标准三阶段流程（信息搜集 → 数据分析 → 报告撰写），为一个产业生成 5 份深度报告，再整合为多标签 HTML 手册。

**适用场景：** 系统学习一个产业，从入门到投资决策。

**流程：**

| 阶段 | 内容 |
|------|------|
| 第一阶段：信息搜集 | 确定产业范围，列出 5 个关键维度，收集公开数据（财报、研报、行业统计） |
| 第二阶段：数据分析 | 整理产业链结构、市场份额、企业对比，提炼核心数字和关键趋势 |
| 第三阶段：报告撰写 | Markdown 撰写 → build_html.py 转 HTML → 5 份合并为单文件手册 |

**手册规范：**
- 封面：标题 + 10 个必记数字卡片
- 5 个标签页，每页对应一份报告
- 支持暗色模式
- 底部互相跳转链接（产业之间交叉引用）
- 商业分析风格，面向小白读者
- 避免偏激用语

**已完成的产业：**

| 产业 | 5 个维度 |
|------|----------|
| 汽车产业 | 全球车企格局 / 中国新能源 / 竞争格局 / 购车指南 / 品牌简史 |
| 半导体产业 | 入门扫盲 / 市场与巨头 / 中国突围 / 汽车芯片 / 中美博弈 |

**触发词：** 产业手册、行业报告、认知手册、生成手册

## 安装

将 `skills/` 目录下的 `.md` 文件复制到目标项目的 `.claude/skills/` 目录即可使用。

```bash
# 单个技能
cp skills/party-meeting-record.md 你的项目/.claude/skills/

# 全部技能
cp skills/*.md 你的项目/.claude/skills/
```

---

## 使用方式

在 Claude Code 对话中直接发送触发词，或使用斜杠命令：

```
/party-meeting-record
/industry-handbook
```
